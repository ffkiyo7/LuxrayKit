# OpenDesign Reference Package

This package contains the prompt and current PWA screenshots for the Pokemon Champions Tool redesign.

Use `PWA_OPENDESIGN_PROMPT.md` as the main instruction text for OpenDesign.

Screenshots:

- `01-team-page.png`: current team page and bottom navigation
- `02-member-editor-sheet.png`: current member editor bottom sheet
- `03-damage-calculator.png`: current damage calculator page
- `04-damage-calculator-expanded-config.png`: damage calculator with ability config expanded
- `05-speed-line-top.png`: current speed line top section
- `06-speed-line-benchmarks.png`: speed line chart and benchmark cards
- `07-rule-dex-list.png`: current rule dex list
- `08-rule-dex-detail.png`: current rule dex detail
- `09-settings-page.png`: current settings page

Design note:

The redesign should keep the existing dark, dense, mobile-first tool feel. The screenshots are for visual continuity; the prompt defines the new information architecture.
