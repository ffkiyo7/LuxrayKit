# 移动端视觉回归

更新时间：2026-06-05

## 策略

- Playwright 使用 390px 移动视口做 smoke 级截图回归。
- 测试使用本机 Chrome channel。
- 视觉测试会阻止 service worker，避免旧缓存影响截图。
- 基线截图位于 `tests/pwa/visual.spec.ts-snapshots/`。

## 覆盖范围

- 环境首页、完整宝可梦榜、宝可梦环境详情。
- 队伍列表、队伍详情。
- 队伍成员编辑与 SP picker。
- 工具聚合页。
- 计算页进攻方 / 防守方选择与配置入口。
- 速度线。
- 图鉴列表、详情、Pokémon 属性筛选、招式属性筛选。
- 图鉴详情的正方形头像、身高 / 体重、特性展开、种族值、属性弱点 / 抵抗 / 免疫、learnset 搜索与排序。
- 我的页本地备份 / 离线缓存入口。

## 命令

```bash
npm run test:visual
npm run test:visual -- --update-snapshots
```

完整 PWA 回归：

```bash
npm run test:pwa
```
